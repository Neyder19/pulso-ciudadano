# Pulso Ciudadano

Portal editorial cívico optimizado para GitHub Pages.

## 🚀 Ejecutar local

Abrir index.html con Live Server o servidor local.

## 🌐 Desplegar en GitHub Pages

1. Subir repositorio
2. Settings → Pages
3. Deploy from main / root

## 🖼 Reemplazar imágenes

Colocar archivos en:

/assets/img/

Preferir formato .webp

## 📊 Actualizar datos

Editar:

/assets/data/articles.json
